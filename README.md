# [WIP] `vt-style`

## Usage

```
$ vt-style help
Usage: vt-style [options] [command]

Options:
  -h, --help                    display help for command

Commands:
  init <file>                   initialize a style JSON
  build <source> [destination]  build a style JSON from the YAML
  help [command]                display help for command
```
